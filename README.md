# Arduino Automation Projects

## About This Repository
Kumpulan proyek otomasi sederhana yang dirancang untuk membantu murid memahami konsep dasar elektronika, sensor, aktuator, dan logika pemrograman melalui penerapan yang dekat dengan kehidupan sehari-hari.

Setiap proyek disusun secara sederhana namun memiliki potensi besar untuk dikembangkan lebih lanjut, sehingga mendorong pembelajaran berbasis praktik, kreativitas, dan pemecahan masalah nyata.

---

## Project Objectives
- Mengenalkan konsep otomasi dan sistem kendali
- Melatih logika berpikir dan pemrograman dasar
- Menghubungkan teknologi dengan konteks kehidupan sehari-hari
- Mendukung pembelajaran berbasis proyek (Project Based Learning / P5 / STEM)

---

## Project List
Repository ini memuat beberapa proyek otomasi, antara lain:

1. **Automatic Hand Wash System**  
   Sistem cuci tangan otomatis menggunakan sensor dan pompa air.

2. **Automatic Trash Bin**  
   Tempat sampah otomatis dengan sensor jarak dan motor servo.

3. **Automatic Clothesline**  
   Jemuran otomatis berbasis sensor hujan.

4. **Soil Moisture Monitoring System**  
   Penyiraman tanaman otomatis berdasarkan kelembapan tanah.

5. **RFID Door Access System**  
   Pintu otomatis menggunakan kartu RFID.

6. **Automatic Gate Barrier**  
   Palang pintu otomatis berbasis sensor ultrasonik.

7. **Automatic Stair Lighting**  
   Lampu tangga otomatis menggunakan sensor infrared.

---

## Hardware Components
Beberapa komponen yang digunakan dalam proyek-proyek ini:
- Arduino Uno / Nano
- Sensor Ultrasonik
- Sensor Infrared (IR)
- Sensor PIR
- Sensor Kelembapan Tanah
- Sensor Hujan
- Modul RFID MFRC522
- Relay 5V
- Motor Servo
- Pompa DC
- LED dan resistor pendukung
